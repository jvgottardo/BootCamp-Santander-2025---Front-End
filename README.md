## 📅 Estrutura do Bootcamp
O bootcamp é composto por:
1. **Módulos teóricos** — fundamentos essenciais.
2. **Exercícios práticos** — fixação do conteúdo.
3. **Projetos finais** — aplicação real dos conhecimentos.
